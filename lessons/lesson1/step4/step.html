Computing Conditions

Boolean values in OCaml can be either true or false. They can be computed by comparing other values. For example:

1 > 2


Comparison operators cannot only be used on numerical values, but on any values, even complex values (you will see them later). For now, we can compare strings:

"aaaaaa" < "bbb"

"3" < "22"


There are different comparison operators, strict ones like < and >, and non-strict ones like <= and >=. For example:

'a' <= 'b'

'b' <= 'b'

'c' <= 'b'


For equality, you can use =, and for inequality, there is <>. For example:

1 = 1

1 <> 1

"1" = 1


Oups, OCaml does not allow you to compare values with different types, so we must be more careful:

"1" = string_of_int 1
